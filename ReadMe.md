<h2>Bootstrap-Paginate<h2>

<h4>How to use</h4>
<span>Basically, the only thing that is required is the placement of the </span>`<div class="pagination" />` and the addition of the following code your page:<br/>
<pre>
<code>
  $(document).ready(function(){
      $('.table').paginate();
  });
</code>
</pre>
